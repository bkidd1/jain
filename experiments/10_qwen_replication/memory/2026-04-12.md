# 2026-04-12 — Qwen Cross-Architecture Replication

## Session Summary

Attempted replication of Gemma-4 KV cache sycophancy findings on Qwen2.5-3B.

### Key Results

1. **Prefill encoding replicates directionally**
   - Clean KV → 80% correct, Hint KV → 45% sycophantic
   - Direction consistent with Gemma-4 despite noise

2. **V-advantage observed but not statistically validated**
   - V-only: 50% coherent-correct
   - K-only: 25% coherent-correct
   - 2× ratio consistent with Gemma-4's pattern
   - But only ~10-12 effective samples per condition

3. **RoPE produces KV patching artifacts**
   - Mismatched prompt lengths → garbled outputs
   - Even matched lengths: only 40-60% coherent
   - This is a methodological finding about technique scope

### Technical Notes

- Qwen2.5-3B shows MORE sycophancy than Gemma-4 (66% vs 40%) with simple prompts
- Chat template activates "correction mode" reducing sycophancy to 10%
- RoPE encodes position into K vectors, breaking cross-prompt patching
- Matched-length prompts mitigate but don't eliminate artifacts

### Paper Implications

- Gemma-4 remains primary result (n=100, validated)
- Qwen provides directional support, not full replication
- Frame RoPE issue as open problem for future work
- Honest about evidence hierarchy
